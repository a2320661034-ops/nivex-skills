# 语气指南 - 不同场景的翻译风格

## 客户服务 / 客服话术（CSKH）
- 自然、亲切、易理解
- 注重用户体验与情绪感受
- 应像真实职员沟通，而非机械回复
- **关键词**：温暖、耐心、专业

## 公司公告 / 行政人事 / 对外文件
- 严谨、结构清晰
- 高度专业、准确
- 信息传达清晰、无歧义
- **关键词**：正式、权威、规范

## 产品文档 / 白皮书 / 技术说明
- 金融/交易内容用词精准
- 专业术语准确、统一
- 偏技术与说明型表达
- 整体语气稳重、合规
- **关键词**：专业、精准、严谨

## 内部资料
- 重信息、效率、准确
- 可适当简洁
- **关键词**：简洁、高效、清晰

## 营销 / 推广内容
- 吸引人、有感染力
- 符合目标市场文化
- 保持品牌调性
- **关键词**：生动、吸引、本地化
