# Crypto 术语表

| 中文 | 越南语标准译法 | 备注 |
|------|--------------|------|
| 加密货币 | Tiền mã hóa / Tiền kỹ thuật số | 禁止用 "Tiền ảo" |
| 区块链 | Blockchain | |
| 交易所 | Sàn giao dịch | 例：Binance, Nivex |
| 智能合约 | Hợp đồng thông minh | Smart Contract |
| 去中心化金融 | Tài chính phi tập trung | 简称 DeFi |
| 钱包 | Ví (tiền điện tử) | |
| 私钥 | Khóa riêng tư | Private Key（极重要） |
| Gas 费 | Phí Gas (Phí giao dịch) | |
| 做多 | Long / Mua (kỳ vọng tăng) | |
| 做空 | Short / Bán (kỳ vọng giảm) | |
| 杠杆 | Đòn bẩy | 例：100x |
| 保证金 | Ký quỹ (Margin) | 开仓押金 |
| 止损 | Cắt lỗ | Stop-loss |
| 止盈 | Chốt lời | Take-profit |
| 爆仓/清算 | Cháy tài khoản / Thanh lý | 币圈常用"俚语" |
| 牛市 | Thị trường bò (Tăng) | |
| 熊市 | Thị trường gấu (Giảm) | |
| 流动性 | Thanh khoản | |
| 链上数据 | Dữ liệu trên chuỗi | 区块链上记录的实际交易 |
| 质押/Staking | Đóng băng / Đặt cọc nhận lãi | 持币获得奖励 |
| 去中心化 | Phi tập trung | 区块链核心特性 |
| 冷钱包/热钱包 | Ví lạnh / Ví nóng | 离线(安全) / 在线(便捷) |
| 法定货币 | Tiền pháp định | 如 VND、USD、CNY |
| 赋能 | Tăng tính ứng dụng | 为 Token 创造实际使用价值 |
| 量化交易 | Giao dịch định lượng | 使用算法和AI自动交易 |
