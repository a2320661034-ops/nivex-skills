# Marketing & BD 术语表

| 中文 | 越南语标准译法 | 备注 |
|------|--------------|------|
| 投流/买量 | Chạy quảng cáo / Đổ traffic | 在FB、Google、X上设置广告 |
| 目标受众 | Đối tượng khách hàng / Tiệp khách hàng | 目标用户分群 |
| 转化率 | Tỷ lệ chuyển đổi | 广告观看者转为注册/充值用户的比例 |
| 获客成本 | Chi phí thu hút một khách hàng | 获得1个新用户的花费 |
| 留存率 | Tỷ lệ giữ chân người dùng | 回到Nivex平台的用户数 |
| 痛点 | Điểm chạm nỗi đau | 产品解决的客户难题 |
| Hook | Câu mồi / Điểm thu hút | 前3秒抓住注意力的内容 |
| 私域流量 | Lưu lượng truy cập nội bộ | Zalo群、Telegram群、Nivex自有社群 |
| 潜在客户 | Khách hàng tiềm năng | 有兴趣但尚未充值/交易的人 |
| 返佣 | Hoa hồng / Hoàn phí | 合伙人/IB邀请新用户的奖励 |
| 话术 | Kịch bản tư vấn / Sale kit | 与客户对话的标准话术 |
| 跟进 | Theo sát / Chăm sóc | 首次接触后持续维护联系 |
| 成交 | Chốt đơn / Khớp lệnh | 客户决定下单投资 |
| 代理/经纪人 | Đại lý / Người môi giới | 为Nivex拓展市场的合作方 |
