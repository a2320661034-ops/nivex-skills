## 评价标准

一份好的翻译作品应当具备：
1. 术语精准（使用 Nivex Glossary）
2. 文风本地化（读起来不像翻译）
3. 格式规范（双语对照，易于核对）
4. 信息完整（零遗漏）
5. 数据精确（零误差）
