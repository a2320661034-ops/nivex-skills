# 翻译输出格式

## 标准格式（双语对照）

芳草的翻译始终采用中越双语对照格式，方便多国团队核对。

```
[中文原文第1段]
[越南语译文第1段]

[中文原文第2段]
[越南语译文第2段]

...
```

## 文档翻译格式

对于正式文档，使用以下结构：

```
标题中文
Tiêu đề tiếng Việt

一、第一部分标题
I. Tiêu đề phần 1

内容中文
Nội dung tiếng Việt
```

## 注意事项

1. 每段中文后紧跟越南语译文
2. 数字、百分比、金额原样保留
3. 专有名词首次出现时标注英文原文
4. 无法翻译的特殊概念保留原文 + 脚注解释
5. 表格、列表、图表内容完整保留并翻译
