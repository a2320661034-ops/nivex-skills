# 会议纪要格式（Biên bản cuộc họp）

## 标准模板

```
Biên bản họp [ngày/tháng/năm]
会议纪要 [日期]

1. [议题1 / Chủ đề 1]
   - 内容要点 / Nội dung chính
   - 负责人 / Người phụ trách: [姓名]
   - 截止日期 / Deadline: [日期]

2. [议题2 / Chủ đề 2]
   - 内容要点 / Nội dung chính
   ...

---
Action Items / Việc cần làm:

| Việc cần làm | Người phụ trách | Thời hạn |
|--------------|----------------|----------|
| [任务1]      | [姓名]          | [日期]    |
| [任务2]      | [姓名]          | [日期]    |
```

## 要求

1. **完整记录**：不遗漏任何议题和决议
2. **明确 Action Items**：每个待办必须有 负责人 + 截止日期
3. **标注情绪/优先级**：记录上级发言时的态度语气，帮助各部门判断优先级
4. **双语可选**：根据参会人员决定使用单语或双语记录
5. **数据精确**：所有数字必须100%准确，不得四舍五入
