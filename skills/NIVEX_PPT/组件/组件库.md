# NIVEX PPT 组件库

所有组件的 HTML 结构和 CSS。生成 PPT 时从这里选取组件组合。

---

## 1. 标准卡片 (card)
用途：通用信息展示

```html
<div class="card animate-in animate-in-delay-N">
  <p style="font-size: 1.3rem; color: var(--white); margin-bottom: 10px;">中文内容</p>
  <p style="color: var(--gray);">Nội dung tiếng Việt</p>
</div>
```

### 带左侧强调线
```html
<div class="card card-highlight animate-in">
  <div class="card-accent"></div>
  <div style="margin-left: 15px;">内容</div>
</div>
```

### 高亮卡片
```html
<div class="card card-highlight animate-in">
  <!-- card-highlight 自带绿色渐变背景 + 绿色边框 -->
</div>
```

## 2. 特性方格 (feature-box)
用途：并列展示 3-5 个特性/能力，带图标

```html
<div class="grid-5">
  <div class="feature-box animate-in animate-in-delay-N">
    <div class="icon">📄</div>
    <div class="cn">中文</div>
    <div class="vn">Tiếng Việt</div>
  </div>
  <!-- 重复 -->
</div>
```

---

## 3. 特性列表 (feature-list + feature-item)
用途：带左侧绿线的纵向列表，适合要点说明

```html
<div class="feature-list">
  <div class="feature-item animate-in animate-in-delay-N">
    <div>
      <div class="cn">中文要点</div>
      <div class="vn">Điểm chính tiếng Việt</div>
    </div>
  </div>
  <!-- 重复 -->
</div>
```

---

## 4. 时间线 (timeline)
用途：历史演进、阶段发展

```html
<div class="timeline">
  <div class="timeline-item animate-in animate-in-delay-N">
    <div class="timeline-dot">年份</div>  <!-- 加 class="active" 高亮当前 -->
    <div class="timeline-content">
      <h3>中文标题 <span style="font-size:0.85rem;color:var(--gray);font-weight:400;">/ Tiêu đề VN</span></h3>
      <p>中文描述 · Mô tả tiếng Việt</p>
    </div>
  </div>
  <!-- 重复，最后一项自动隐藏连接线 -->
</div>
```

---

## 5. 对比卡片 (comparison)
用途：旧 vs 新、前 vs 后

```html
<div style="display: grid; grid-template-columns: 1fr 80px 1fr; gap: 20px;">
  <!-- 左侧（旧） -->
  <div class="card comparison-card animate-in animate-left">
    <h3>旧方式 / Cách cũ</h3>
    <ul class="comparison-list">
      <li><span class="cn">中文</span><span class="vn">Tiếng Việt</span></li>
    </ul>
  </div>

  <!-- 箭头 -->
  <div class="arrow-between animate-in animate-scale">→</div>

  <!-- 右侧（新） -->
  <div class="card comparison-card new card-highlight animate-in animate-right">
    <div class="card-accent"></div>
    <h3>新方式 / Cách mới</h3>
    <ul class="comparison-list">
      <li><span class="cn">中文</span><span class="vn">Tiếng Việt</span></li>
    </ul>
  </div>
</div>
```

## 8. 误区纠正 (myth-row)

```html

    <div class="myth-card wrong">
      <div class="icon">❌</div>

    </div>
    <div class="myth-arrow">→</div>
    <div class="myth-card right">
      <div class="icon">✓</div>
      <div class="cn">正确观点</div>
      <div class="vn">Quan điểm đúng</div>
    </div>
  </div>

</div>
```

## 12. 大脑/核心概念区 (brain-section)
用途：核心概念展示，带大图标

```html
<div class="brain-section animate-in">
  <div class="brain-icon">🧠</div>  <!-- 可换任何 emoji -->
  <div class="brain-text">
    <h2>核心概念 <span style="font-size:1.3rem;font-weight:400;">/ Khái niệm cốt lõi</span></h2>
    <p style="color: var(--gray);">描述 · Mô tả</p>
  </div>
</div>
```

自定义图标背景色：
```html
<div class="brain-icon" style="background: linear-gradient(135deg, #4a9eff 0%, #2d7dd2 100%);">📦</div>
```

## 17. 分栏布局 (split-layout)

## 布局网格快速参考

| 类名 | 列数 | gap | 用途 |
|------|------|-----|------|
| grid-2 | 2 | 30px | 左右对比、双栏 |
| grid-3 | 3 | 20px | 三列卡片 |
| grid-5 | 5 | 15px | 图标方格 |
| position-grid | 5 (flex) | 15px | 岗位卡片，末行居中 |
| award-grid | 3 | 20px | 奖项卡片 |
| split-layout | 1fr 350px | 40px | 左内容右面板 |
