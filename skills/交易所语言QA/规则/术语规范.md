# 术语规范 — 术语标准表构建与维护

## 目的
确保交易所界面中的核心术语翻译统一、准确、符合行业标准。
术语表是QA审查的基石——**没有术语标准，就没有修正依据，禁止进入全量扫描。**

## 铁律
- **无术语表 = 阻断全量扫描**。必须先完成术语表构建（含行业对标）才能开始审查。
- **行业对标是必须步骤**，不是可选步骤。必须用 WebSearch/WebFetch 抓取头部交易所一手数据，不允许凭经验猜测。
- 术语表构建的4步流程必须严格按序执行，不允许跳过行业对标直接扫描。

---

## 术语标准表结构

每个语言的术语表文件（如 `术语表/越南语.md`）采用统一结构：

| 字段 | 说明 |
|------|------|
| 序号 | 自增编号 |
| 分类 | 7大分类之一 |
| 中文术语 | 源语言术语 |
| English | 英文参考 |
| 标准翻译 | 确认后的标准目标语言翻译 |
| 出现次数 | 在CSV中出现的频次 |

### 7大分类
1. 交易术语（现货、合约、杠杆、仓位等）
2. 订单相关（委托、成交、限价、市价等）
3. 账户资产（充值、提现、余额、钱包等）
4. 产品功能（理财、跟单、定投、借贷等）
5. 安全认证（KYC、2FA、密码、验证等）
6. 界面通用（首页、设置、搜索、通知等）
7. 代理返佣（返佣、邀请、层级等）

## 构建流程（4步）

### Step 1: 提取高频术语
- 从所有CSV中提取源语言短文本（语言标识列值 ≤ 8个字符）
- 统计每个短文本的出现频次
- 按频次降序排列，取 Top N（建议 ≥ 500条）
- 同时提取长文本中的高频中文关键词

### Step 2: 行业对标研究（必须，不可跳过）

**此步骤是强制性的。** 无论时间压力如何，都必须执行行业对标。不允许凭经验猜测术语翻译。

参照来源（按优先级）：
1. **Binance** 目标语言界面（binance.com/{语言代码}）
2. **OKX** 目标语言界面（okx.com/{语言代码}）
3. **Bybit** 目标语言界面（bybit.com/{语言代码}）
4. **Gate.io** 目标语言界面（gate.io/{语言代码}）

执行方法：
- 用 **WebSearch** 搜索「{交易所名} {目标语言} 交易 界面/教程/使用指南」
- 用 **WebFetch** 抓取交易所目标语言页面，提取导航栏、功能模块、按钮文案
- 重点覆盖：交易术语、订单相关、账户资产、产品功能 四大核心分类

对每个术语：
- 查看4家交易所的翻译
- 记录共识度（4家一致 > 3家一致 > 2家一致）
- 取共识度最高的翻译作为建议标准
- **如果4家都没有该术语，则标记为"需龙老师确认"，不自行编造**

### Step 3: 人工确认（关键步骤）
- 将术语对照表提交龙老师审核
- 标注"需确认"的项目：行业无共识、或当前翻译与建议标准有明显差异
- 龙老师确认/覆盖的术语标记为"龙老师覆盖"，优先级最高
- **绝不跳过此步骤**

### Step 4: 固化入术语表文件
- 写入 `术语表/{语言名}.md`
- 后续QA直接加载此文件作为标准
- 包含龙老师覆盖区域（单独列出，最高优先级）

---

## 术语匹配逻辑

执行QA扫描时，按以下顺序匹配：

1. **龙老师覆盖** — 最高优先级，精确匹配
2. **语言标识列精确匹配** — lang_key 完全等于术语表中的中文术语
3. **源语言短文本精确匹配** — 简体中文列值 ≤ 8字符，完全等于术语表中的中文术语
4. 匹配成功 → 对比目标语言列值与标准翻译
5. 不一致 → 标记 TERMINOLOGY_MISMATCH

---

## 维护规则

1. **只增不删** — 即使某术语不再使用，也保留记录（用于历史追溯）
2. **龙老师覆盖永远优先** — 行业标准与龙老师覆盖冲突时，用龙老师的
3. **增量更新** — 每次QA迭代如有新术语发现，追加入术语表
4. **版本记录** — 术语表头部记录版本和更新日期

---

## 新语言术语表创建模板

创建新语言术语表时：
1. 复制 `术语表/越南语.md` 为 `术语表/{新语言}.md`
2. 更新元信息（语言名称、CSV列名、语言代码）
3. 清空术语数据和覆盖列表
4. 按上述4步流程重新构建
5. 保留文件结构不变（元信息 → 覆盖区 → 完整术语表 → 片段映射）
