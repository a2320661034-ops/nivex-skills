### 语境判断规则（关键！）

**同一个越南语词在不同语境含义不同，必须用中文源文判断：**

#### "sao chép" 的语境判断
- 中文含 `跟单|带单|复制交易|Copy Trade` → **Copy Trade语境** → sao chép → Copy Trade
- 中文含 `复制|粘贴|私钥|地址|助记词|复制成功|已复制` → **复制文本语境** → sao chép 保留不动
- 规则：先判断语境，再替换

#### "đơn hàng" 的语境判断
- 中文含 `委托|下单|挂单|订单|撤单|限价|市价` 且**不含** `购买|付款|OTC|法币|商家|买币` → **交易语境** → đơn hàng → lệnh
- 中文含 `购买|付款|OTC|法币|商家|买币|卖币|充值|提现` → **OTC买币语境** → đơn hàng 保留不动
- ⚠️ 注意："hóa đơn"（发票/账单）中的"đơn"不能被替换！

#### "nhà giao dịch" 的语境判断
- 中文含 `交易员|交易达人|带单` 且**不含** `OTC|商家` → Trader
- 中文含 `OTC|商家` → nhà giao dịch OTC 保留不动

### 越南语大小写规则

1. **句首大写**：每个独立句子/条目的第一个字母必须大写
2. **数字后时间单位不大写**：`1 ngày`（✅），`1 Ngày`（❌）
3. **多行文本续行不强制大写**：`\n` 后如果是语义延续，不必大写
4. **专有名词保持原样**：Futures, Spot, Copy Trade, Trader, AI, KYC 等
