# 术语对照表输出格式

## 文件规范
- **文件名**：`中{目标语言简称}术语对照表.csv`（如：中越术语对照表.csv、中韩术语对照表.csv）
- **编码**：UTF-8 with BOM (utf-8-sig)
- **引用**：QUOTE_ALL

---

## 列定义

| 列名 | 说明 |
|------|------|
| 序号 | 自增 |
| 分类 | 7大分类之一 |
| 中文术语 | 源语言术语 |
| English | 英文参考 |
| 当前翻译(所有版本) | CSV中出现的所有目标语言翻译版本 |
| 建议标准翻译 | 基于行业研究的建议标准 |
| 是否需确认 | 是/否 |
| 出现次数 | 在CSV中的频次 |

---

## 分类体系

固定7类：
1. 交易术语
2. 订单相关
3. 账户资产
4. 产品功能
5. 安全认证
6. 界面通用
7. 代理返佣

---

## 人工审核交付格式

当需要提交龙老师审核时，输出 `.xlsx` 格式：
- **需确认的行**：黄色高亮
- **已有标准的行**：无高亮
- 龙老师修改后的行标记为"龙老师覆盖"

审核完成后，将确认结果固化入 `术语表/{语言名}.md`。
