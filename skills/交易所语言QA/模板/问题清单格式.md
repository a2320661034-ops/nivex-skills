# 问题清单输出格式

## 文件规范
- **文件名**：`{目标语言}问题清单.csv`
- **编码**：UTF-8 with BOM (utf-8-sig)
- **引用**：QUOTE_ALL

---

## 列定义

| 列名 | 说明 | 示例 |
|------|------|------|
| 序号 | 自增，从1开始 | 1 |
| 来源 | 文件标识 | APP / H5 / Web / 代理后台 |
| 编号ID | 原CSV中的编号ID | 7701 |
| 优先级 | P0 / P1 / P2 | P0 |
| 问题类型 | 类型名称，多类型用"/"分隔 | CONTAINS_CHINESE |
| 语言标识 | 原CSV中的语言标识列值 | 跟单体验券允许用户... |
| 当前翻译 | 目标语言列当前值 | Phiếu trải nghiệm...失效 |
| 建议修正 | 完整的修正后文本（空=需人工翻译） | Phiếu trải nghiệm...hết hiệu lực |
| 确认 | 留空，供人工填写 | |
| 人工修正 | 留空，供人工覆盖 | |

---

## 排序规则
1. **优先级**：P0 → P1 → P2
2. **来源**：APP → H5 → Web → 代理后台
3. **编号ID**：降序（最新的在前）

---

## 控制台摘要

扫描完成后输出：

```
═══════════════════════════════════════
{目标语言}翻译质量扫描摘要
═══════════════════════════════════════
总问题数：N

按优先级：
  P0（致命）：N
  P1（严重）：N
  P2（建议）：N

按问题类型：
  EMPTY：N
  UNTRANSLATED_COPY：N
  CONTAINS_CHINESE：N
  FULLWIDTH_PUNCTUATION：N
  TERMINOLOGY_MISMATCH：N
  ...

按来源：
  APP：N
  H5：N
  Web：N
  代理后台：N

问题清单已输出：{文件路径}
═══════════════════════════════════════
```
